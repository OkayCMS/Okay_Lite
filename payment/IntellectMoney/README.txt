1. Скопировать папку IntellectMoney в каталог сервера http://ваш_сайт/payment/
2. В админ. панели добавить новый способ оплаты, в качестве обработчика указать IntellectMoney, выбрать валюту, заполнить "ID магазина" и "Секретный ключ", сохранить изменения.
3. В ЛК IntellectMoney настроить магазин на прием платежей по протоколу IntellectMoney, в качестве Result Url указать http://ваш_сайт/payment/IntellectMoney/callback.php
